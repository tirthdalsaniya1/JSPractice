**Git command and steps used to push the code and create the repo**

1) git init
2) git status
3) git add .
4) git commit -m "Initial commit: Organize JavaScript learning project into folders"

Create a new repository on GitHub:
Go to https://github.com/new
Name it (e.g., JSPractice or JavaScript-Learning)
Do NOT initialize with README, .gitignore, or license
Click "Create repository"
Once created, copy the repository URL (HTTPS or SSH)

5) git remote add origin https://github.com/tirthdalsaniya1/JSPractice.git
6) git branch -M main
7) git push -u origin main
